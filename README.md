# Addons for World of Warcraft (classic)

## Craft Usage Tooltip

A lightweight World of Warcraft addon that displays the professions using an ingredient directly inside the item tooltip.

Get it on CurseForge: https://www.curseforge.com/wow/addons/craft-usage-tooltip

## Wisdom of Azeroth

Wisdom of Azeroth is a lightweight World of Warcraft flavor addon that displays a random Azeroth-themed proverb when you log in.

Get it on CurseForge: https://www.curseforge.com/wow/addons/wisdom-of-azeroth

## Gasp Of Pandaria

Un mini‑jeu de réflexion pour World of Warcraft, inspiré du jeu Gasp : cliquez sur une gemme pour retourner les huit voisines… mais pas celle que vous avez cliquée.  
Le but : obtenir une grille uniforme en un minimum de coups.
